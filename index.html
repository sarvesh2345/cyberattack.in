Cyber Attacks.

- A cyber attack is an exploitation of computer systems and networks.It uses malicious code to alter computer code,logic or data and lead to cybercrimes ,such as information and identify theft.
- Cyber attacks can be classified into the following categories: 1. Web-based attacks 2.System-based attacks.

Web-based attacks:

- this are the attacks which occurs on a website or web applications .some of the impoertant web-based attacks are as
1. Injection attacks : It is an attack in which some data will be injected into a web applications to manipulate the applications to manipulate the applications and fetch the required information.
2. Session Hijacking: It is a security attack on a user session over a protected network.Web applications create cookies to store the state and user sessions .By stealing the cookies , an attacker can have access to all of the user data.
3. Phishing: Phishing is a type of attacks which attempts to steal sensitive information like user login credentials and credit card number.It occurs when an attackers is masquwrading as a trustworthy entity in electronics communication.
4. Denial of service: It is an attack which meant to make a server or network resourse unavailable to the users.It accomplishes this by flooding the target with traffic or senidng information that triggers a crash.

System-based attacks:

- These are the attacks which are intended to compromise a computer or a computer network.some of the important system-basrd attacks are as follows:
1. Virus: It is a type of malicious software program that spread throughtout the computer files without the knowledge of user.It is a self-replicating malicious computer program that replicates by inserting copies of itself into another computer programs when executed.
2. Worms: It is atype of malware whose primary function is to replicates itself to spread to uninfected computers.It works same as the computer virus.
3. Trojan horse : It is a malicious program that occurs unexpected changes to computer settings and unusal activity , even when the computer should be idle. It misleads the user of its true intent.
